## ADDED Requirements

### Requirement: Color system established
The site SHALL use a defined color system with warm cream/parchment backgrounds and dark green, burgundy, and muted gold accents.

#### Scenario: Brand colors applied consistently
- **WHEN** any page of the site renders
- **THEN** the color palette uses warm cream/parchment for backgrounds, dark green/burgundy/gold for accents, and maintains consistent color usage across components

### Requirement: Typography hierarchy
The site SHALL establish a clear typography hierarchy with elegant serif headings and readable body text.

#### Scenario: Headings use serif fonts
- **WHEN** any heading displays on the site
- **THEN** the heading uses an elegant serif typeface appropriate for The Divvy's nostalgic aesthetic

#### Scenario: Body text is readable
- **WHEN** body text displays
- **THEN** the font is highly readable for long-form content with appropriate size and line height

### Requirement: Spacing system
The site SHALL use generous whitespace and a consistent spacing system.

#### Scenario: Consistent spacing applied
- **WHEN** components and sections render
- **THEN** spacing follows a consistent scale providing visual breathing room and hierarchy

### Requirement: Antique-inspired aesthetic
The site SHALL incorporate subtle antique-paper or auction-catalogue inspired design elements.

#### Scenario: Visual atmosphere matches brand
- **WHEN** a visitor views any page
- **THEN** the design evokes antiques shops, auction rooms, and faded catalogues through textures, borders, or subtle decorative elements

### Requirement: Soft cards and shadows
The site SHALL use soft card styling and subtle shadows rather than harsh borders.

#### Scenario: Cards feel warm
- **WHEN** cards display (episode cards, content blocks)
- **THEN** they use soft shadows, subtle borders, and warm backgrounds

### Requirement: Avoid trendy UI gimmicks
The site SHALL avoid generic SaaS design, harsh whites, neon colors, over-designed layouts, and excessive animations.

#### Scenario: Design feels timeless
- **WHEN** any page renders
- **THEN** the design avoids trendy modern UI patterns in favor of classic, warm, literary aesthetic
